# Installation

## Prerequisites

- Python 3.8 or higher
- pip package manager
- Git

## Clone the Repository

```bash
git clone https://github.com/francescacraievich/mola-adversial-nsga2.git
cd mola-adversial-nsga2
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Verify Installation

```bash
python -c "import numpy; import scipy; print('Installation successful!')"
```

## MOLA SLAM Setup

For MOLA SLAM integration, additional setup steps are required. See the MOLA documentation for details.

## Isaac Sim Setup

This project requires NVIDIA Isaac Sim for simulation and ground truth generation. Refer to the [Isaac Sim documentation](https://docs.omniverse.nvidia.com/isaacsim/latest/index.html) for installation instructions.
